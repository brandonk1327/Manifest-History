About Manifest History

Goals

Our goal of this project is to creative an interactive map to assist people with learning history as it happened. The user will be able to interact with the map using pins that can be activated by hovering or clicking. Our current span of history only includes the 19th century and a limited amount of pins per year.

We do not own any of the information or historical texts that are displayed on our webpage, but are just acting as a gateway for people to easily view and explore it. Since this website is a project for our Sophomore Project class at Oakland University our site is not monetized and is for our educational purposes only.

How it works

This project was coded primarily to be web based. Every functionality of the project works through the user's browser. This makes it easily accessible to anyone that would like to use it. It can also be run client sided, because the code is mostly client side. It was coded in HTML and JavaScript. HTML (along with CSS and Bootstrap) being the code that makes the webpage look nice, adds hyperlinks, and gives structure to the page. JavaScript adds the functionality of the web app that the user will be going through.

The project was tracked and version controled through the use of git; using GitHub also to provide the project on the web.
